# Отчет по этапу 2.6.1

## Сделано

- Исправлен manifest основной колоды: 124 физические карты, 26 беспределов по 1, 18 одиночных не-беспределов по 1 и 40 уникальных карт по 2.
- Добавлена команда `python -m src.game.validate_manifest` с JSON-сводкой и ошибками по физическому составу колоды.
- Убраны конфликтующие поля `GameConfig` для тиражей служебных стопок; manifest остается единственным источником этих количеств.
- Расширен targeting: выбранный колдун, каждый колдун, правый/левый враг, правый или левый враг, самый хилый/могучий и враги хилее/могучее игрока.
- Групповая атака легенды теперь целит всех колдунов и идет от следующего игрока по кругу.
- Стартовые карты закрыты для strict: `Знак`, `Пшик`, `Палочка`; у `Палочки` реализован добор 2 карт только при убийстве целью этой атаки.
- Парсер простых эффектов поддерживает больше базовых шаблонов: мощь, добор, лечение, урон, простой сброс карт, выдачу вялой палочки.
- `effect_coverage` использует тот же full-card анализ, что и strict, и показывает `implemented_with_tests >= 12`.
- Добавлена команда `python -m src.game.replay_summary <replay.json>` для компактной диагностики replay.
